What is FIS-Designer?

FIS-Designer is a hobby project aiming to help design dashboards for the fantastic FIS-Control MMI device.

How to use

Left hand side - current config:

  • These values represent the FIS-Control config 1:1.

Right hand side - visual preview of your configuration and two toolbars:

  • Lower toolbar allows you to test the design dynamically.
  • Upper toolbar:
    • Opens the image manager.
    • Allows loading and saving of the designs.
 More help
  • Look at the tooltips that appear when you hover over some elements, they contain important information.
  • For more detailed explanation of the values please consult the official FIS-Control manual.
