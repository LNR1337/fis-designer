What is FIS-Designer?

FIS-Designer is a hobby project aiming to help design dashboards for the fantastic FIS-Control MMI device.

Help
  • Look for the tooltips that appear when you hover over some elements, they contain important information.
  • For more detailed explanation of the config values please consult the official FIS-Control manual.
